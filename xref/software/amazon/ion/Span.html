20  /**
21   * An immutable reference to a consecutive sequence of values (perhaps
22   * including large hierarchies) within some base source of Ion data.
23   * <p>
24   * <b>WARNING:</b> This interface should not be implemented or extended by
25   * code outside of this library.
26   * <p>
27   * A span is conceptually comprised of two abstract <em>positions</em> or
28   * offsets within the base source.  The <em>start</em> position denotes the
29   * leftmost edge of the span, and the <em>finish</em> position denotes the
30   * rightmost edge.
31   * Positions lie <em>between</em> values, and when the start and finish
32   * positions are equal, the span is said to be <em>empty</em>.
33   * <p>
34   * A span is said to <em>cover</em> the values that lie within its edges.
35   * <p>
36   * A span is <em>balanced</em> if it starts and finishes within the same
37   * container, otherwise it is <em>unbalanced</em>.  This library currently
38   * does not support unbalanced spans.
39   * <p>
40   * Since different source types require different positioning techniques,
41   * spans is {@link Faceted} to expose the position implementation.
42   *
43   * <h2>Acknowledgements</h2>
44   * This design and terminology is heavily based on Wilfred J. Hansen's work on
45   * subsequence references.
46   *
47   * @see SpanProvider
48   * @see Spans
49   * @see TextSpan
50   * @see OffsetSpan
51   *
52   * @see <a href="http://portal.acm.org/citation.cfm?id=133234">Subsequence
53   * References: First-Class Values for Substrings</a>
54   *
55   */
56  public interface Span
57      extends Faceted
58  {
59      /**
60       * Gets a span covering the container of this span.
61       *
62       * @throws IonException if the current span is at top-level.
63       */
64  //    public Span containerSpan();  // TODO later.  Move to a facet?
65  
66  
67      /**
68       * Gets a span covering all the children of this span, which must cover a
69       * single container.
70       *
71       * @throws IonException if this span covers anything other than a
72       * single container.
73       */
74  //    public Span contentSpan();  // TODO later.  Move to a facet?
75  }
